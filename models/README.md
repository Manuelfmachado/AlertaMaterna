# Directorio para modelos entrenados

Los modelos se guardan aquí después de ejecutar `src/train_model.py`:

- `modelo_mortalidad_xgb.pkl` - Modelo XGBoost para predicción de mortalidad
- `scaler_mortalidad.pkl` - Scaler para normalización de features
- `umbral_riesgo_obstetrico.pkl` - Umbrales para clasificación de riesgo
- `umbral_mortalidad.pkl` - Umbral de alta mortalidad

**Nota:** Los archivos .pkl no se suben a Git por su tamaño.
